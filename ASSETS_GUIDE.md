## Image Guidelines

1. **Quality**: Use high-resolution images (minimum 1200px width for hero/feature images)
2. **Optimization**: Compress images for web (use tools like TinyPNG, ImageOptim, or Next.js Image optimization)
3. **Format**: Prefer JPG for photos, PNG for graphics with transparency
4. **License**: Ensure you have proper licensing rights for all images
5. **Alt Text**: All images have descriptive alt text already implemented

## Adding Images

1. Place images in the `/public` folder
2. Use the exact filenames specified above
3. The components will automatically use these images
4. If an image is missing, components will gracefully fall back to default displays
